# Blank repo
## About this repository
This repository has been composed based on the [embassy repository]. That repository is about 10GB and contains far more than you need to get started on an STM32H755ZI-Q.
This repository contains all the necessary elements to start using embedded Rust and embassy on the STM32.
It contains a simple blinky script, and some default dependencies.
While much is possible with these basic dependencies, it might still be an incomplete list.
